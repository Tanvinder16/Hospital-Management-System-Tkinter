# Hospital Management System

A desktop-based Hospital Management System developed using Python, Tkinter, and Pandas.

## Features

- Login Authentication
- Patient Registration
- Store Patient Records in Excel
- Search Patient by ID
- Analytics Dashboard

## Technologies Used

- Python
- Tkinter
- Pandas
- OpenPyXL

## Project Workflow

Login → Dashboard → Add Patient → Save Data → Search Patient → Analytics

## Future Improvements

- Update Patient Records
- Delete Patient Records
- Appointment Booking
- Advanced Analytics

This project was developed as part of a Python Workshop.
